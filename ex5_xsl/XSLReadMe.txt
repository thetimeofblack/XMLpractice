xsl is a kind of document based on the html document. It can transfer xml document into html document for data presentation by using xsl document type definition 

xslt <template>
xslt <value-of>
xslt <for-each>
xslt <sort>
xslt <if>     
<xsl:if test="expression">
...如果条件成立则输出...
</xsl:if>
xslt <choose>

= （等于）
!= （不等于）
&lt; （小于）
&gt; （大于）
